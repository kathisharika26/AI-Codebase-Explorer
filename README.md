# 🥈 AI Codebase Explorer

## 📌 Project Overview

AI Codebase Explorer is an AI-powered web application that allows users to upload a project as a ZIP file, extract its contents, and explore the project structure through an interactive interface. The long-term goal is to integrate AI features that can analyze code, explain project architecture, detect bugs, and generate documentation.

---

## 🚀 Features

### ✅ Completed

* User Interface with React + TypeScript
* FastAPI Backend
* ZIP File Upload
* ZIP File Validation
* Upload Progress Bar
* Extract ZIP Files
* Display Uploaded Project Files
* REST API Integration
* Git & GitHub Integration

### 🔄 In Progress

* VS Code-style File Explorer
* Folder Tree View
* File Preview
* Search Files

### 🎯 Planned Features

* AI Code Explanation
* AI Bug Detection
* AI Documentation Generator
* AI Code Summarization
* Project Architecture Visualization

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/kathisharika26/AI-Codebase-Explorer.git
```

### Backend

```bash
cd backend

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt

python -m uvicorn app.main:app --reload
```

### Frontend

```bash
cd frontend

npm install

npm run dev
```
